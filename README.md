## VexalUI - An Inspired UI Library
This UI library was inspired by WindUI and is in the work for its design and functionality.
Currently supports: 
- Creating Windows
- Creating Tabs
- Creating Buttons
- Creating Toggles
- Creating Sliders
- Creating Colorpickers (in progress)
